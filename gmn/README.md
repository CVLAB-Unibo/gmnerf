# Graph Metanetworks for Processing Diverse Neural Architectures (ICLR 2024)

![Teaser](https://research.nvidia.com/labs/toronto-ai/GMN/images/paper_figs/gmn_fig1.jpg)

This directory contains a subset of the [official implementation](https://github.com/cptq/graph_metanetworks) of the ICLR 2024 [paper](https://arxiv.org/abs/2312.04501) "Graph Metanetworks for Processing Diverse Neural Architectures" by Lim et al., with some modifications aimed at integrating our `hash table → graph` conversion into the original Lim et al.'s codebase.
