# Deep Learning on Object-centric 3D Neural Fields (TPAMI 2024)

![Teaser](https://cvlab-unibo.github.io/nf2vec/img/teaser.png)

This directory contains a subset of the [official implementation](https://github.com/CVLAB-Unibo/nf2vec) of the TPAMI 2024 [paper](https://arxiv.org/abs/2312.13277) "Deep Learning on Object-centric 3D Neural Fields" by Zama Ramirez et al., with some modifications aimed at integrating tri-planar and hash-based NeRFs into the original Zama Ramirez et al.'s codebase.
